# MyMICDS-Discord-Bot
This is the official bot for MyMICDS

# Running
To install dependencies, do

```npm install -D```

and then to run a compilation of it, run

```npm run compile-run```

# Production
To compile for production (a clean build essentially), do

```npm run compile:prod```
